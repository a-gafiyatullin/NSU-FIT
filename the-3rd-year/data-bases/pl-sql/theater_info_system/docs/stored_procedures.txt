Требуется реализовать следующие хранимые процедуры БД:
    - вставки во все таблицы;
    - получение полной информации (все поля) о спектаклях, соответсвующих запросу (критерии запроса - форма Спектакли.Информация);
    - получение полной информации (все поля) о ролях и актерах на этих ролях для конкретного спекталя;
    - получение полной информации (все поля) о музыкантах для конкретного спекталя;
    - получение требований на роль для конкретного спектакля;
    - получение всех актеров, подходящих на роль для конкретного спектакля;
    - получение всех дат показа конкретного спектакля;
    - получение списка для выпадающих списков:
        + дирижеров постановщиков;
        + жанров
        + возрастных категорий;
        + режиссеров постановщиков;
        + художников постановщиков;
        + авторов;
        + стран;
        + характеристик;
        + категорий работников;
        + уровней образования;
        + полов;
        + званий;
        + конкурсов;
        + типов постановщиков;
        + музыкальных инструментов;
        + дат показа спектакля из репертуара.
    - получение полной информации (все поля) об авторах, соответсвующих запросу (критерии запроса - форма Авторы.Информация);
    - получение полной информации (все поля) о работниках, соответсвующих запросу (критерии запроса - форма Работники);
    - получение полной информации (все поля) об актерах, соответсвующих запросу (критерии запроса - форма Актеры.Информация + Информация о наградах актеров);
    - получение полной информации (все поля) о ролях конкретного актера (критерии запроса - форма Актеры.Информация + Информация о ролях актеров);
    - получение полной информации (все поля) о постановщиках, соответсвующих запросу (критерии запроса - форма Постановщики.Информация);
    - получение полной информации (все поля) о спектаклях конкретного постановщика (критерии запроса - форма Постановщики.Информация + Спектакли постановщика);
    - получение полной информации (все поля) о музыкантах, соответсвующих запросу (критерии запроса - форма Музыканты.Информация);
    - получение полной информации (все поля) о спектаклях конкретного музыканта (критерии запроса - форма Музыканты.Информация + Спектакли музыканта);
    - получение полной информации (все поля) о гастролях, соответсвующих запросу (критерии запроса - форма Гастроли.Информация);
    - получение полной информации (все поля) о доступных билетах на спектакли, соответсвующие запросу (критерии запроса - форма Билеты и абонементы. Информация - билеты);
    - получение полной информации (все поля) о доступных абонементах, соответсвующих запросу (критерии запроса - форма Билеты и абонементы. Информация - абонементы);
    - получение полной информации (все поля) о билетах в конкретном абонементе;
    - получение полной информации (все поля) об абонементе для конкректного билета;
    - получение кол-ва проданных билетов на спектакли, соответсвующие запросу(критерии запроса - форма Экономические показатели);
    - получение суммарной выручки от проданных билетов на спектакли, соответсвующие запросу(критерии запроса - форма Экономические показатели).
